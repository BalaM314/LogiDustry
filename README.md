# Mindustry Logic

This is a repo for logic scripts I make in [Mindustry](https://github.com/Anuken/mindustry).

## My Mlog standards

Use `__<name>` for temporary variables
Use `_<name>` for jumps

**Temp Variables**

- `__reg<n>`
- `__stack<n>`

**Naming conventions**

Use camel case for naming convention.
